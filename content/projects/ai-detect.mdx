# AI Detect

Overview and architecture of the AI Detect project. Built with TensorFlow.js for real-time detection.

## Architecture

- **Frontend**: Next.js, React, Tailwind CSS
- **ML**: TensorFlow.js for browser-based inference
- **Features**: Video feed detection, recording, screenshots
